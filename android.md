# Android Studio

* Mac OSX
* Android Studio 3.0.0-4

## ショートカット

* Completion
  * Ctrl + Space は Spotlight で使っているので変更
    * Spotlightはアプリの切り替えに使っているので必要
    * IdeaVIMを入れているのでVimっぽくしたいので `C-x C-o` とした
      * `C-x C-o` はオムニ補完
* Layoutの design と text の切り替え
  * `Shift + Ctrl + ← or →` 
* ファイラ
  * `Shift` を連打
* エラー解決
  * `alt + Enter`
  * `fn` を押す必要があって面倒だからやめたい（悩み中...）
